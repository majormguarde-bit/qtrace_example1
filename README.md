# QTrace Monitor

Система мониторинга и прогнозной аналитики состояния роторного оборудования.

## Описание
QTrace Monitor — это веб-приложение на базе Flask, предназначенное для визуализации телеметрии и раннего обнаружения неисправностей. Система включает в себя:
- **Эмулятор ротора**: Генерация данных вибрации, температуры и оборотов (RPM) в различных режимах (Норма, Дисбаланс, Дефект подшипника).
- **Ядро аналитики QTrace**: 
  - Фильтр Калмана для очистки сигнала от шумов.
  - Контрольные карты Шухарта (правило 3σ) для детекции аномалий.
  - SOM-like классификация состояний.
  - **Прогнозный модуль RUL (Remaining Useful Life)**: Оценка остаточного ресурса узла на основе тренда аномальности.

## Структура проекта
- `qtrace_app/` — основной пакет приложения.
  - `api/` — API эндпоинты для управления режимами и получения данных.
  - `core/` — алгоритмы обработки сигналов и эмуляции.
  - `main/` — роуты для отображения интерфейса.
  - `services/` — фоновые службы (генерация телеметрии).
  - `static/`, `templates/` — фронтенд часть (Bootstrap 5, Chart.js).

## Установка и запуск

1. Клонируйте репозиторий:
   ```bash
   git clone https://github.com/majormguarde-bit/qtrace_example1.git
   cd qtrace_example1
   ```

2. Создайте виртуальное окружение и установите зависимости:
   ```bash
   python -m venv venv
   source venv/bin/activate  # Для Windows: venv\Scripts\activate
   pip install -r requirements.txt
   ```

3. Запустите приложение:
   ```bash
   python app.py
   ```
   Приложение будет доступно по адресу: `http://127.0.0.1:5000`

## Использование
- Используйте кнопки управления режимами для изменения состояния эмулятора.
- Наблюдайте за графиком вибрации: зеленый коридор отображает допустимые границы 3σ.
- Следите за карточкой **Прогноз (RUL)**: она покажет расчетное время до выхода оборудования из строя при текущем тренде.
